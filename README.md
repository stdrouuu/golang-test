## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/golang-test.git
    cd golang-test
    ```

2. **Run the application**
    ```bash
    go run main.go
    ```

## Requirements

- [Go](https://golang.org/dl/) 1.18 or higher
